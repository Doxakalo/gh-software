Gonector Release Notes

This is a brief history of important changes across released versions.

2.0.1

  • Support for Apple Silicon/M1 processors

2.0

  • Added support for Infinite Peripherals DPP receipt printers
  • Added support for CAEN skID, a new high-performance RAIN/UHF RFID reader
  • Added support for Alien ALR-S350 hand-held RAIN/UHF RFID reader
  • Added support for Zebra CS4070 and DS3678 barcode scanners
  • Additional Zebra scanners may be compatible but have not been tested
  • Extended CAEN RFID readers support by enabling user memory reading and writing
  • Extended iPhone built-in NFC reader support to enable NFC tag writing
  • Added compatibility with FileMaker iOS App SDK 19
  • Extended Infinite Peripherals Infinea X support to allow for triggering scripts by pressing programmable buttons
  • Added possibility to control visual, audible and haptic feedback for readers which support it
  • Improved cadence of script triggers for devices able to read data in large quantities (scales, RFID readers)

1.2.2

  • Updated license handling/and activation for eSellerate end-of-life
  • Minor performance and FileMaker 18 compatibility improvements

1.2

  • Support for Infinea mPos flat and Infinea mPos BP500 terminals
  • Support for Ingenico Link/2500
  • New example for Ingenico Link/2500 with CSOB application
  • New example for Infinea mPos terminals

1.1

  • Plug-in binaries for Mac, Windows, and iOS are now code-signed
  • Plug-In renamed to Gonector
  • Support for Custom Printers
  • New Custom Printer example
  • Support for Scales connected via BlueSnap Smart adapter
  • New Scales example
  • iPhone native NFC Reader
  • New NFC Reader example
  • Linea and Infinea devices
  • New Linea Infinea example
  • New qIDmini example

1.0

  • Scan-Go Plug-In created
  • Supports qIDMini R1170 RFID reader

Connecting the BlueSnapAdapter

BlueSnap Smart - Bluetooth 4.0 (BLE) to RS232 Adapter daily use connect

  • Make sure the scale is switched off and connected through the RS-232 cable to the BlueSnap Smart device
  • Make sure that the iOS device has Bluetooth switched on
  • Switch the adapter on by pressing the power button shortly
  • The LED light should be blinking green
  • Use the function Gonector_Connect to connect to the device
  • After couple of seconds the LED should be steadily on
  • BlueSnap Smart device has been connected
  • Switch the scale on
  • Every time the scale sends data the LED light should blink blue

Note: To ensure a successful connect to the BlueSnap Smart device the scale must be switched off.
Note: To send RawCommands directly to the scale it is necessary to have the scale switched on.
The issue is that when the scale sends data into the BlueSnap Smart device the plugin is not able to connect to it.
To ensure that the scale does not send any data to the device it is necessary to switch the scale off.

If the name of the device is not known you can use the function Gonector_Get( "devices" ; "blueSnapAdapter" ) to find out the name of the device.
The scales themselves must be set up in plugin with a "scaleReadCommand" and "scaleWeightPattern". To set these you need to use function Gonector_Set.

Do not pair the device through iOS settings.

BlueSnap Smart - Bluetooth 4.0 (BLE) to RS232 Adapter initial setup

  • Disconnect any device connected to the adapter
  • Make sure that the adapter is fully charged
  • Make sure that the iOS device has Bluetooth switched on
  • Switch the adapter on by pressing the power button shortly
  • Get device name with function Gonector_Get( "devices" ; "blueSnapAdapter" )
  • Use the function Gonector_Connect to connect to the device
  • Use the function Gonector_Set with selector "baudrate" and newValue "9600"
  • Wait until the LED control light indicates to be switched off (there is no activity)
  • Wait at least 10 more seconds
  • Device has been set up and is ready to be switched on and connected

Scales Hardware Setup

LESAK J1-RWP - Gonector_SetScriptEvent setup (needs to be manual)

  • Press "UNIT" and "PRINT(M+)" simultanously
  • Press "TARE" button until "F4 Prt" is reached
  • Press "ZERO"
  • Press "TARE" until the wished option is reached:
    On stable"St1", "StC" or "StAB"
    Continous"Cont"
  • Press "ZERO" if 9600 baud rate is visible. Or press "TARE" until 9600 is reached
  • Press "UNIT"

LESAK J1-RWP - Gonector_Read setup (needs to be manual)

  • Press "UNIT" and "PRINT(M+)" simultanously
  • Press "TARE" button until "F4 Prt" is reached
  • Press "ZERO"
  • Press "TARE" until "Ask" option is reached
  • Press "ZERO" if 9600 baud rate is visible. Or press "TARE" until 9600 is reached
  • Press "UNIT"

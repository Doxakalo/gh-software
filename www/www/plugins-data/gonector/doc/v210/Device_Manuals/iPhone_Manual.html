Manual for iPhone (NFC reader)

Limitations

  • Supported are iPhones from version 7 and higher.
  • The iPhone must have at least iOS 11.
  • The RFID card to be read must be properly formatted and non-empty. You can read more about NDEF format here.

Solving issues with iPhone as a device

  • I cannot connect to the iPhone. It says that device is not available
    iPhone uses a special library called Core NFC. To enable this library it is necessary to go to your project in Xcode, select the target you build, go to tab Capabilities and switch on the Near Field Communication Tag Reading.

Manual for ALR-S350

Connecting ALR-S350

  1. Turn on the ALR-S350. The bluetooth indicator will start blinking (blue led).
  2. Use function Gonector_Get("devices" ; "AlienReader") to find out the name of your device.
    It should return string like "EXA51 ABCDEF1234567"
  3. Use function Gonector_Connect with the name of your device. When device connects the bluetooth indicator will turn on and stop blinking.

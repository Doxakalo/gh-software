Manual for qID R1170

This document describes known limitations of RFID reader qID R1170 as well as measured distances for reading on certain types of materials.
This document is only overview of capabilities qID R1170 RFID reader. For more information see reader manual.

Connecting RFID reader

Connecting RFID reader qID R1170 has to be done in iOS settings, section Bluetooth. In Bluetooth devices, you should see name of reader (similar to qID R1170) and description of status. In order to device work properly, the status should be "Connected". Once connected light on RFID reader will start glowing blue (left LED).

Limitations of RFID reader

Following list summarises knowledge gathered by measurement performed by our tester team:

  • When more RFID tags are present in reading zone of device, reader can create non-existing value
  • Material on which RFID tag is placed and its position is important for reading. For example: RFID tag placed on plastic bag placed vertically can be read from distance approximately 15cm, however the same tag placed on wood or glass can be read from distance approximately 50cm
  • Speed of reading: one tag can be read by reader approximately 200 per minute, 3 tags can be read by tag 175 per minute
  • Reader can connect to iOS device from distance max. 10m
  • Reader will lose its connection to iOS device from distance at least 15m


Following table shows distances for RFID reader for different sides of device. Front side is the one with display, buttons and LEDs.

Solving issues with RFID

This section describes issues with reader, which can occur together with possible solution.

  • Reader is connected to iOS, but I cannot read tags nor get firmwareVersion
    RFID reader uses special protocol for communication, which has to be specified in your app. Into the .plist file of the project in Xcode, add new Row with the value "Supported external accessory protocol" and add the Strings:
    • it.caenrfid.protocol
  • Readers display shows the same value no matter if tags are present or not
    1. Turn off the RFID reader
    2. Shut down the application
    3. Turn device on and reconnect it to the iOS device
    4. Start your application again

Apple App Store Requirements

If you build an app for qIDmini device and are going to distribute your app on the App Store, you will need to register the app with CaenRFID in order to obtain MFi Authorization

To do so you have to send email to info@caenrfid.com with the following information:

  • bundle name
  • bundle display name
  • bundle identifier
  • version number
After few days CaenRFID will provide you with an identification number (PPID) that you have to include in review notes when submitting your app

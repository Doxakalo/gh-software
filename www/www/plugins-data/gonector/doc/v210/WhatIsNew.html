What's new in Gonector 2.1

The version 2.1 brings support for Zebra ZQ630 RFID printer

The following improvements have been introduced:

  • Added support for Zebra ZQ630
  • Additional Zebra RFID printers may be compatible but have not been tested
  • Fixed custom script steps
  • Fixed inline documentation
  • Minor performance and FileMaker 19 compatibility improvements

For more information

Please, refer to the following places to learn more about Gonector:

  • If you are new to FileMaker plug-ins, start by checking the UsingFMPlugIns.html file inside the Documentation folder. It briefly explains what plug-ins are, why they are useful, and how they are commonly used.
  • Basic instructions for installing the plug-in, system requirements, quick start guide, usage terms, troubleshooting tips, and other useful information can be found in the ReadMe.html file inside the Documentation folder.
  • A complete history of past versions of Gonector is located in the ReleaseNotes.html file
    inside the Documentation folder.
  • If you decide to use Gonector you should also read the 24U Software License Agreement.pdf file inside the Documentation folder. Don't worry about language of this file. It is intentionally brief and simple so that an average user understands under what conditions Gonector may be used.
  • Finally, the HowToRegister.html file inside the Documentation folder will help you to understand what different licensing options are available for Gonector and how to purchase them.

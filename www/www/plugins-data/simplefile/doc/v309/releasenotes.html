24U SimpleFile Plug-In Release Notes

This is a brief history of important changes across released versions.

3.0.9

  • Added Hosting friendly function

3.0.8

  • Fixed SFile_Version function returning wrong result

3.0.7

  • Fixed SFile_Decompress that was not properly decompressing folders in a zip archive on MacOS system

3.0.6

  • Support for Apple Silicon/M1 processors

3.0.5

  • Updated license handling/and activation for eSellerate end-of-life
  • Minor performance and FileMaker 18 compatibility improvements

3.0.4

  • Plug-in binaries for Mac and Windows are now code-signed

3.0.3

  • Fixed issue causing crash on macOS 10.13 High Sierra

3.0.2

  • Improved support for reading binary files using SFile_Read function
  • Fixed registration issue on Windows 10
  • Fixed interoperability between SFile_Write with "base64" encoding and FileMaker Base64Encode function

3.0.1

  • Mac OS X: Minimum OS X version updated to 10.8
  • Mac OS X: Fixed SFile_Compress issue that caused generation of corrupted archives
  • Mac OS X: Register dialog now support Copy/Paste

3.0

  • Support for 64-bit FM Server was added
  • Compatibility with IWP and WebDirect was greatly improved
  • SFile_Choose dialog now supports more options
  • SFile_Open enables to select what application will open the file
  • New function SFile_Reveal opens the file in enclosing folder
  • New function SFile_Run can run an application from the solution with user given arguments
  • Added zip functionality - SFile_Compress and SFile_Decompress functions
  • Exif rotation width and height mismatch removed
  • Fixed RTF underlining bug, supported underline formats are single and double. RTF text shading is ignored. (Those are FM limitations)
  • Fixed a few bugs regarding empty and non-existent files
  • Fixed SFile_Write from container bug
  • FileMaker path works correctly with mounted disks. (It prefers folder in root directory in case of name collision)
  • Fixed timestamps' precision

2.0

  • Formatted text can be saved or loaded to / from RTF file
  • Ability to resize and rotate picture files
  • Mount and unmount network volumes
  • Base64 encoding support
  • SFile_Copy now preserves timestamps
  • Relative path works consistently with an opened solution
  • Containers with links work correctly
  • No limit for long unix paths
  • Recorded sound can be written to file
  • Works correctly with FileValut protected folders
  • Several problems with paths fixed
  • Works on FileMaker Pro 11 Advanced natively, compatible with older versions down to 8.5

1.1.1

  • Fixed bug with Simple Pack serial numbers
  • Documentation update

1.1

  • Can read and write container pictures and files into FileMaker Pro containers
  • Works on FileMaker Server 9 in Instant Web Publishing
  • Works on FileMaker Pro 9 Advanced natively

1.0.2

  • No limit in reading files
  • Removed limit in the directory listing

1.0.1

  • Runs natively on both PowerPC and Intel based Macs
  • Works with FileMaker Pro 7.0, 8.0 and 8.5

1.0

  • Initial version release

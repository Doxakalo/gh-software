Supported Devices

Supported Arduino boards

  • Arduino UNO board and its 100% compatible clones

Arduino UNO information

  • Output Voltage: 5V
  • Input Voltage: 7-12V
  • Number of digital inputs/outputs: 14
  • Number of PWM outputs: 6
  • Number of analog inputs: 6
  • EEPROM memory: 1KB
  • Flash memory: 32KB
  • SRAM memory: 2KB
  • Frequency: 16MHz

Supported Accessories

  • Arduino Starter Kit
  • Arduino Ethernet shield
  • esp8266esp-01 wi-fi module

Bridge for Phidgets Release Notes

This is a brief history of important changes across released versions.

4.0

  • Support for FileMaker Server
  • Support for Humidity Sensor
  • Support for Distance Sensor
  • Support for Sound Sensor
  • Support for Accelerometer
  • Support for Gyroscope
  • Support for Magnetometer
  • Support for Spatial
  • Support for Current Input
  • Support for Capacitive Touch
  • Support for Encoder
  • Support for Pressure Sensor
  • Support for Frequency Counter
  • Support for Resistance Input
  • Minor performance improvements

3.1.5

  • Support for Apple Silicon/M1 processors

3.1.4

  • Support for TextLCD adapter
  • Fixed special characters on LCD Display
  • Minor performance and FileMaker 19 compatibility improvements

3.1.3

  • Updated license handling/and activation for eSellerate end-of-life
  • Minor performance and FileMaker 18 compatibility improvements
  • FileMaker crashing when disconnecting an SBC fixed

3.1

  • Plug-in binaries for Mac and Windows are now code-signed
  • RFID tags locking
  • PhidgetRFID Read-Write (1024) fully tested
  • Returning "?" in case of error fixed
  • Support for iOS

3.0

  • Added support for SBC4 device
  • Added support for VINT devices
  • Added support for Stepper controller
  • Added compatibility with new Phidget drivers (Phidget22)
  • Added compatibility with FileMaker 16 and 17
  • All functions return "?" in case of error. You can use Phidget_Get("lastError") for more detailed error description.

2.0.1

  • Fixed registration issue on Windows 10
  • Improved registration dialogs

2.0

  • 64-bit support on Windows and Mac OS X
  • Added compatibility with FileMaker 14 and 15
  • Added support for Phidgets connected over network
  • Added support for PhidgetRFID Read-Write, including RFID tag writing
  • Added support for PhidgetBridge
  • Added support for PhidgetAnalog
  • Added support for PhidgetTextLCD
  • Added support for PhidgetAdvancedServo 8-Motor
  • Threshold parameter in function Phidget_SetScriptEvent works correctly now
  • Improved registration dialogs
  • On advanced Servo controller you can set Max and Min value

1.0

  • Support for all Phidgets from Phidget Starter Kit 1
  • Simple interface to control all Phidgets
  • Scripts can be used to event handling

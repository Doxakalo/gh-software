24U Phone Companion Release Notes

This is a brief history of important changes across released versions.

3.0.2

  • Updated license handling/and activation for eSellerate end-of-life
  • Minor performance and FileMaker 18 compatibility improvements

3.0.1

  • Plug-in binaries for Mac and Windows are now code-signed

3.0

  • The minimum supported version of FileMaker Pro is now 16
  • Some plugin functions can be now called as FileMaker Script Steps
  • Phone_Get("lastErrorMessage") now displays messages correctly
  • Time out error message improved.
  • Added support for VOIP systems using SNOM phones, successfully tested with RingCentral
  • Distribution Archive now includes the latest TeamCall 3.8.19
  • The 24U Phone Companion now supports more PBX models and phones!
